# 二维码条形码生成工具

一个功能强大的二维码和条形码批量生成工具，支持多种打印尺寸，可直接部署到 GitHub Pages。

## 功能特性

- ✅ **双模式支持**：支持条形码（CODE128）和二维码生成
- ✅ **批量生成**：支持一次输入多个条码/二维码数据（每行一个，按Enter切换）
- ✅ **多种尺寸**：支持 4" x 6" (100x150mm) 和 4" x 2" (100x50mm) 两种打印尺寸
- ✅ **文本输入**：支持通用标题和额外文本输入，一并打印
- ✅ **优化布局**：针对不同尺寸自动调整布局，确保打印效果完美
- ✅ **实时预览**：输入数据后实时预览生成效果
- ✅ **简洁UI**：现代化的用户界面，操作简单直观

## 使用方法

1. 选择生成模式（条形码或二维码）
2. 选择纸张尺寸（4x6 或 4x2）
3. 输入通用标题（可选，会显示在每个标签顶部）
4. 输入额外文本（可选，会显示在每个标签底部）
5. 在文本框中输入条码/二维码数据，每行一个，按Enter切换下一行
6. 实时预览生成效果
7. 点击"打印标签"按钮进行打印

## 部署到 GitHub Pages

### 方法一：直接使用 GitHub Pages

1. 将 `index.html` 文件上传到你的 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择主分支（main/master）作为源
4. 访问 `https://你的用户名.github.io/仓库名/` 即可使用

### 方法二：使用 gh-pages 分支

1. 将 `index.html` 推送到 `gh-pages` 分支
2. GitHub 会自动部署，访问 `https://你的用户名.github.io/仓库名/` 即可

## 技术栈

- **HTML5** + **CSS3**
- **Tailwind CSS** - 现代化UI框架
- **JsBarcode** - 条形码生成库
- **QRCode.js** - 二维码生成库

## 浏览器支持

- Chrome/Edge（推荐）
- Firefox
- Safari
- 其他现代浏览器

## 打印设置建议

- **4" x 6" 标签**：适合快递单、大标签等场景
- **4" x 2" 标签**：适合货架标签、小标签等场景

打印时请确保：
- 打印机支持自定义纸张尺寸
- 在打印设置中选择正确的纸张尺寸
- 关闭页边距（margin: 0）
